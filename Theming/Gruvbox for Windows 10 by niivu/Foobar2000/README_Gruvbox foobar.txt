:: Gruvbox foobar2000 by niivu ::

Requirement:
:: foobar2000 http://www.foobar2000.org/download

Setup:
:: copy contents into your foobar folder.
:: Make sure your runnung the latest foobar with ColumnsUI as interface
:: go to preferences -> ColumnsUI -> Main -> use Import Button to import the Gruvbox .fcl of your choosing.

Best view:
:: disable toolbars (click top left edge button) and status bar.
